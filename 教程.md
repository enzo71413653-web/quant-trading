## 1. 量化到底在干什么（30 秒建立认知）

一句话：**把"我觉得这样买卖能赚钱"的直觉，变成能用历史数据检验的规则。**

一个闭环：

```
数据  →  想法(策略)  →  回测  →  看结果  →  改进
 ↑                                          │
 └──────────────  纸面循环，不上真钱  ────────┘
```

这三个工具正好对应前三步：**akshare 拿数据 → backtesting 回测想法 → quantstats 评估**。

## 2. 工具一：akshare —— 拿数据（`01_get_data.py`）

**跑法**：`python scripts\01_get_data.py`
**它做了什么**：把沪深300ETF(510300)的日线行情拉下来，存成 `data/510300.csv`。

**关键代码在讲什么**：
```python
SYMBOL = "510300"          # ← 改这里换标的（6位代码）
df = ak.fund_etf_hist_em(symbol=SYMBOL, period="daily",
                         start_date=START, end_date=END, adjust="qfq")
```
- `adjust="qfq"` = **前复权**。股票分红/拆股会让价格"跳空"，前复权把历史价格按现在口径拉平，**回测必须用复权价**，否则会凭空算出假的涨跌。
- 返回的是中文列名，脚本已帮你改成 `Open/High/Low/Close/Volume`（回测/分析都认这套英文名）。

**换成你自己的标的**：
| 你想看 | 改成 |
|---|---|
| ETF（如沪深300ETF 510300、纳指ETF 513100） | `ak.fund_etf_hist_em(symbol="510300", ...)`（当前脚本） |
| 个股（如贵州茅台 600519） | 把那行换成 `ak.stock_zh_a_hist(symbol="600519", period="daily", start_date=START, end_date=END, adjust="qfq")` |

**真实输出**（末尾 3 行，价格是前复权后的）：
```
2026-07-08  Open 4.836  Close 4.798  High 4.872  Low 4.793  Volume 4846848
```
拿到干净、连续、复权的数据，就算过了第一关。

## 3. 工具二：quantstats —— 给持仓体检（`02_portfolio_checkup.py`）

**跑法**：`python scripts\02_portfolio_checkup.py`（要先跑过 01）
**它做了什么**：把收盘价变成日收益率，算出关键风险/收益指标，并生成一份 HTML 报告到 `reports/`。

**真实结果 + 每个数字什么意思**（沪深300ETF 2019→2026）：

| 指标 | 数值 | 怎么读 |
|---|---|---|
| 累计收益 | **+99.5%** | 这 7 年多价格大约翻倍 |
| 年化波动 | **21.0%** | 波动越大越颠簸。21% 属于股票型的正常水平 |
| 最大回撤 | **−44.7%** | 从最高点最惨跌了近一半 → **问自己：真跌 45%，我拿得住吗？** |
| 夏普比率 | **0.56** | 每承担 1 单位风险换来的超额收益。经验：>1 不错，0.5 左右一般，<0 别碰 |

> **最大回撤是新手最该盯的数字**。很多人只看收益，结果半路被 −45% 吓得割在最低点。先看你扛不扛得住，再看能赚多少。

**怎么体检你自己的持仓**：把 `SYMBOL` 换成你持有的标的分别跑；想看整个组合，就把多只的收益按你的仓位权重加权求和成一条 `returns` 再喂给 quantstats（这一步想做时找我，给你加个多标的版）。

**报告怎么看**：打开 `reports/510300_report.html`，重点看"累计收益曲线"和"回撤曲线（水下图）"——回撤图能直观看到你会有多长时间在亏钱。

## 4. 工具三：backtesting.py —— 验证一个想法（`03_backtest_sma.py`）

**跑法**：`python scripts\03_backtest_sma.py`（要先跑过 01）
**它做了什么**：用"双均线交叉"策略在历史上模拟买卖，给出这套规则到底赚不赚。

**策略逻辑（读懂 Strategy 类就够）**：
```python
class SmaCross(Strategy):
    n1, n2 = 20, 60           # 20日短均线、60日长均线
    def init(self):           # 开场算一次指标
        self.s1 = self.I(SMA, self.data.Close, self.n1)
        self.s2 = self.I(SMA, self.data.Close, self.n2)
    def next(self):           # 每根K线到来时做决策
        if crossover(self.s1, self.s2):   # 短线上穿长线 = 金叉 → 买
            self.buy()
        elif crossover(self.s2, self.s1): # 短线下穿长线 = 死叉 → 清仓
            self.position.close()
```
- `init()` 只在开头跑一次，负责算指标；`next()` 每天跑一次，负责"今天买/卖/不动"。
- 这是量化策略最基础的骨架，以后你所有想法都往 `next()` 里塞判断。

**真实回测结果（重点教学）**：

| 指标 | 数值 | 说明 |
|---|---|---|
| 策略收益 Return | **−9.9%** | 这套规则 7 年**亏了 9.9%** |
| Buy & Hold Return | **+40.7%** | 同期"买了就一直拿着"**赚 40.7%** |
| Sharpe Ratio | **−0.10** | 负的 = 承担了风险还倒亏 |
| 胜率 Win Rate | **28.6%** | 21 笔交易只有 6 笔赚 |
| Commissions | **$3875** | 频繁买卖，手续费吃掉一大块 |

> **这就是最值钱的一课**：一个听起来很合理的"金叉买、死叉卖"策略，回测里**又亏钱、又跑输躺着不动、还白交了手续费**。
> 大多数想当然的策略都是这个下场。**先回测，你就能在不亏真钱的情况下发现"这想法不行"。**
>
> （`print(stats)` 里 `_equity_curve`、`_trades` 那两行是内部明细数据，忽略即可。）
> （小注：backtesting 的 Buy&Hold 是"整数股+剩余现金"口径，quantstats 的 +99.5% 是纯价格复利口径，所以两处"持有收益"数字不同——**搞清每个数字怎么算的，本身也是一课**。）

**自己动手改**：
- 改 `n1, n2`（比如 5/20、10/30）再跑，看结果怎么变——你会发现"调参数"很容易骗自己（见第 6 节过拟合）。
- 打开 `reports/510300_backtest.html` 看买卖点画在 K 线上的位置。

## 5. 完整工作流：把三步串成一次真正的"研究"

1. **取数**：`01` 拉 510300。
2. **认识它**：`02` 体检 → 知道它长期 +99.5% 但会 −44.7% 回撤。
3. **提个想法并验证**：`03` 跑双均线 → 发现 −9.9%，**打不过买入持有**。
4. **下结论**：这个策略不行 → 换思路（加止损？换指标？多标的轮动？），回到第 3 步再验证。

**这套"想法 → 回测 → 证伪 → 再想"的循环，就是量化研究的全部。** 赚钱的策略是这样一遍遍试出来的，不是拍脑袋拍出来的。

## 6. 新手必知的坑（这些能让回测骗你）

1. **未来函数 / 前视偏差**：用了当时还不知道的信息（比如用当天收盘价决定当天开盘买）。回测暴赚、实盘暴亏。
2. **过拟合**：把 `n1/n2` 调到历史最赚 = 只是背下了过去，对未来无效。**参数别调太多、要留一段数据做验证。**
3. **忽略交易成本**：看那 $3875 手续费——频繁交易的策略常常是给券商打工。
4. **幸存者偏差**：只测活到今天的股票，会高估收益（退市的都没算进去）。
5. **复权**：不用复权价，分红除权会被当成暴跌，结论全错。
6. **回测赚钱 ≠ 实盘赚钱**：滑点、流动性、情绪、黑天鹅，回测都给不了。回测能做的是**排除坏想法**，不是保证赚钱。
